数据采集说明

方式 1: 浏览器端立即更新

点击上方"立即更新数据"按钮，从你的浏览器直接调用 GitHub API。适合快速验证，免费额度 60 次/小时，可能受速率限制。

方式 2: GitHub Actions 定时任务（推荐）

代码已配置 .github/workflows/fetch-skills.yml，UTC 0 点自动跑一次。点击上方 "GitHub Actions" 按钮可手动触发 workflow，认证后可达 5000 次/小时。

方式 3: 本地命令行

克隆代码后执行：
cd github-skills-trending && node scripts/fetch-skills.js
需要设置环境变量 GITHUB_TOKEN 以获得更高 API 额度。

方式 4: Vercel/CloudStudio Cron

部署后可设置定时任务，每天 0:00 触发数据刷新接口。
